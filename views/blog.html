Pinoko Blog

About

學習前端技術以來開始逐漸了解並學會使用HTML、CSS、JavaScript...做出網頁，畢竟網頁前端設計資訊量爆炸，在學習階段常常搜尋相似的功能，因此希望可以把常用到的概念整理成自己的筆記，這是做了這個blog的其中一個原因。 好用的blog站有很多，利用wordpress架站也不失為一個快速方便的管道，但既然學了網頁設計乾脆就自己從頭開始寫。blog會用到大量站內連結以及許多重複內容（例如首頁的文章標題、標籤），因此我藉由這個blog來練習node.js,ejs,express, RESTful API 等技術。

Technical Skills

  • JavaScript: 站內搜尋
  • Node.js: RESTful API
  • Express: Node.js 中的 Web 開發框架，有完整的 API Reference，可以快速地搭建一個完整功能的網站。
  • Heroku: 架設node.js server
  • jQuery: 動畫特效
  • Bootstrap: 排版、RWD設計
